# 基于Python的网上商城系统 python829

## 项目概述

本项目是基于Python开发的网上商城系统，采用前后端分离的技术架构，前端使用Vue框架，后端使用Django Python框架，数据库选用MySQL。系统功能完整，界面美观大方，代码注释详细，易于维护和理解。

## 技术栈

- 前端：Vue
- 后端：Django Python
- 数据库：MySQL
- 开发工具：PyCharm

## 功能模块

### 系统角色

系统分为管理员和普通用户两种角色。

### 前台功能

- 商品搜索：支持关键词搜索，快速定位商品。
- 商品展示：以列表或卡片形式展示商品信息。
- 商品详情：提供商品图片、描述等详细信息。
- 购物车：管理待购商品。
- 下单结算：支持多种支付方式。
- 个人中心：管理个人信息、收货地址、订单等。

### 管理后台功能

- 个人中心：管理个人信息，修改密码。
- 用户管理：管理用户数据，包括增删改查。
- 商品管理：管理商品数据，包括增删改查。
- 订单管理：管理订单数据，包括增删改查。
- 数据销量分析：查看平台总销量及月销量信息。
- 系统管理：查看系统信息，进行更新维护。
